touch 생성할 파일 이름.형식 : 파일 생성
mkdir :퐅더 생성
	-p : 중간 단계 폴더 자동 생성
rmdir : 디렉터리 삭제
	-p : 디렉터리 삭제시 상위 폴더가 비는 경우 같이 삭제
cd : 디렉터리 이동
ls: 디렉터리 내용 출력
	-a: 숨긴파일 모두 출력
	-d 자체 정보 출력
rm : 파일이나 폴더 삭제
	-i : 삭제 여부 확인
sublime text (한글 지연입력 문제 있음)
	설치 : https://www.sublimetext.com/
	라이선스 비 구매자의 경우 stable로 다운
	subl . : 해당 폴더에 있는 모든 파일 켜기
	subl 파일 명 : 해당 파일 켜기
